# Antagohutch Slack Bot

This is just a bot for being a jerk to Jake on Slack.

## Setup

1. Clone the repository.
2. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```
3. Activate the virtual environment:
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Set up your environment variables (e.g., Slack token, app token).
6. Run the bot!
